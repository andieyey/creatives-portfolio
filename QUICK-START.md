# 🎨 Creative Portfolio Template - Quick Start Guide

## How to Use This Template

This portfolio template is ready to use! Just follow these 3 simple steps:

### Step 1: Download
Download all files and keep them in the same folder:
- index.html
- styles.css
- script.js
- **config.js** (this is where you customize everything!)
- README.md

### Step 2: Customize Your Information

Open **config.js** in any text editor (Notepad, VS Code, etc.) and edit:

```javascript
// Change your personal info
name: "Your Name",              // Replace with your name
title: "Your Title",            // Your professional title
email: "your.email@example.com" // Your email

// Add your projects
projects: [
    {
        title: "My Amazing Project",
        category: "Web Design",
        image: "images/project1.jpg",  // Add your image
        link: "project-link.html"
    }
]
```

### Step 3: Add Your Images

1. Create an `images` folder next to your HTML file
2. Add your photos and project screenshots
3. Update the image paths in config.js

### Step 4: Open and View

Double-click **index.html** to open in your browser. That's it! ✨

✅ Social media links

## 🎨 Changing Colors (Optional)

In config.js, find the colors section:

```javascript
colors: {
    primary: "#6366f1",    // Change to your brand color
    secondary: "#8b5cf6",  // Accent color
}
```

Use a [color picker tool](https://htmlcolorcodes.com/) to find your colors.

## 🚀 Publishing Your Portfolio

### Option 1: Free Hosting (Recommended)

**Netlify Drop** (Easiest):
1. Go to [Netlify Drop](https://app.netlify.com/drop)
2. Drag your entire folder
3. Get a free website instantly!

**GitHub Pages** (If you have GitHub):
1. Create a repository
2. Upload your files
3. Enable Pages in Settings

### Option 2: Your Own Hosting
Upload all files to your web hosting via FTP.

---

## ❓ Need Help?

- **No images showing?** Make sure your images are in the `images` folder
- **Changes not appearing?** Refresh your browser (Ctrl+F5 or Cmd+Shift+R)
- **Want more projects?** Just copy a project object in config.js and add more!

---

## 📱 Testing on Mobile

After customization, test on your phone:
1. Open your browser's developer tools (F12)
2. Click the mobile icon
3. Test different screen sizes
