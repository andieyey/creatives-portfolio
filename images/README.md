# Images Folder

Place your images here:

- **profile.jpg** - Your profile photo for the About section
- **project1.jpg** - Your first project screenshot
- **project2.jpg** - Your second project screenshot
- ... and so on

## Recommended Image Sizes

- **Profile Photo**: 300x300 pixels (square)
- **Project Images**: 800x600 pixels (landscape) or any 4:3 ratio

## Tips

- Use compressed/optimized images for faster loading
- Supported formats: JPG, PNG, WebP
- Keep file sizes under 500KB each
